## Plan: Email Agent Integration

### Goal
Add an email agent that can read your Gmail/Outlook inbox and draft replies for you. Triggered via chat commands only.

### Scope
- **Providers:** Gmail + Outlook
- **Capabilities:** Read inbox + draft replies (no auto-send)
- **Trigger:** Chat command only

### Architecture

#### 1. New Model: ConnectedAccount
**File:** `backend/models/models.py`

Stores OAuth tokens for email providers:
- `user_id` -> users.id
- `provider`: "gmail" | "outlook"
- `account_email`: str
- `access_token`: encrypted via Fernet
- `refresh_token`: encrypted
- `expires_at`: datetime
- `scopes`: List[str]
- `is_active`: bool

#### 2. OAuth Flow
**Files:** `backend/routers/integrations.py` (new), frontend settings page

- **Gmail:** Google OAuth 2.0 with `gmail.readonly` + `gmail.compose` scopes
- **Outlook:** Microsoft Graph OAuth with `Mail.Read` + `Mail.Send` scopes
- Backend routes:
  - `GET /integrations/{provider}/auth-url` ‚Äî returns OAuth URL
  - `GET /integrations/{provider}/callback` ‚Äî handles callback, stores tokens
  - `DELETE /integrations/{provider}` ‚Äî disconnect account
- Frontend settings page adds "Connected Accounts" section

#### 3. Email Service Layer
**File:** `backend/services/email.py` (new)

Functions:
- `fetch_recent_emails(user_id, provider, limit=20)` -> List[Email]
- `draft_reply(user_id, email_id, tone, content)` -> Draft
- `send_draft(user_id, draft_id)` -> bool (requires explicit approval)

- **Gmail:** Google API Client (`google-api-python-client`) + `aiogoogle`
- **Outlook:** `httpx` + Microsoft Graph API
- **Token refresh:** auto-refresh on 401, update DB

#### 4. New Agent: `email_agent.py`
**File:** `backend/agents/email_agent.py`

System prompt: read user's recent emails and draft replies. NEVER send without user approval.

Capabilities:
- `read_inbox` ‚Äî fetch recent emails, summarize by priority/sender
- `draft_reply` ‚Äî draft a reply to a specific email
- `summarize_thread` ‚Äî summarize a conversation thread

#### 5. Supervisor Routing
**File:** `backend/agents/supervisor.py`

Add email keywords: `email`, `inbox`, `reply to`, `draft email`, `gmail`, `outlook`
Route to `email` agent.

#### 6. Frontend Chat Integration
**File:** `frontend/app/(app)/chat/page.tsx`

- Email agent messages show special "Email Card" UI
- Drafted emails render with recipient, subject, body
- Send/Edit/Discard action buttons
- Email actions create notifications
